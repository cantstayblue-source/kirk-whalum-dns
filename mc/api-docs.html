🤖 Agent Operations

MC.agent
MC.agent.myTasks([status])
Get all Valarnette tasks. Optional status filter: 'todo', 'in-progress', 'done'
MC.agent.addTask(title, opts)
Quick-add a task. Options: {description, priority, status, assignee, dueDate}
MC.agent.startTask(id)
Move a task to in-progress status
MC.agent.completeTask(id)
Mark a task as done
MC.agent.nextTask()
Get the highest priority pending task assigned to the agent
MC.agent.log(message)
Log agent activity to the audit trail
MC.agent.statusReport()
Full system status report including workload breakdown for agent and Kirk
MC.agent.triggerEzraEffect(showId)
Create post-show follow-up tasks (Day 1: Thank you, Day 2: Share content, Day 3: Next invite)
MC.agent.scheduleContent(id, date)
Schedule content for a specific publishing date. Sets status to 'ready'
MC.agent.advanceGuest(id, status)
Move a guest through the CRM pipeline. Statuses: identified, reached-out, in-talks, confirmed, appeared, follow-up
Examples
MC.agent.myTasks('todo')
MC.agent.nextTask()
MC.agent.statusReport()
MC.agent.addTask('Review content pipeline', {
  priority: 'high',
  description: 'Audit all draft content and move ready items forward'
})

📡 Events

MC.on / MC.off / MC.once
MC.on(event, handler)
Subscribe to events. Returns an unsubscribe function you can call later
MC.off(event, handler)
Remove a specific event handler
MC.once(event, handler)
One-time subscription — automatically unsubscribes after first trigger
Available Events
change content:created content:updated content:deleted shows:created shows:updated shows:deleted guests:created guests:updated guests:deleted tasks:created tasks:updated tasks:deleted funnels:created funnels:updated funnels:deleted campaigns:created campaigns:updated campaigns:deleted products:created products:updated products:deleted settings:changed data:imported data:reset *
Examples
// Subscribe to new tasks
const unsub = MC.on('tasks:created', task => {
  console.log('New task:', task.title)
})
// Listen to all changes (wildcard)
MC.on('*', (data, event) => {
  console.log('Event:', event, data)
})
// One-time listener for next content creation
MC.once('content:created', item => {
  console.log('First new content:', item.title)
})

📈 Dashboard

MC.dashboard
MC.dashboard.summary()
Aggregate metrics across all collections — content count, upcoming shows, guests, open tasks, content by status, guests by status
MC.dashboard.activity(n)
Recent activity feed. Pass a number to limit results (default: 10)
MC.dashboard.health()
Health score (0–100) with warnings for pipeline bottlenecks and workload issues
Examples
MC.dashboard.summary()
MC.dashboard.activity(5)
MC.dashboard.health()
