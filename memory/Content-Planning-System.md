## 📹 SPLIT-TESTABLE VIDEO SCRIPTS

## 🎬 PRODUCTION NOTES FOR KIRK

**Recording Setup:**
- Phone vertical (9:16) for TikTok/Reels
- Phone horizontal (16:9) for YouTube/Facebook
- Good lighting (window light or ring light)
- Clean audio — use external mic if possible
- 3-5 takes of each section

**A/B Testing Strategy:**
1. Record HOOK A + BODY A + CTA A
2. Record HOOK B + BODY A + CTA B
3. Post both, see which performs better
4. Use winning combo next week
