## 1. THE EZRA EFFECT (Post-Show Relationship Engine)

### How It Works
- **Trigger:** Calendar event (Show Date) ends
- **Wait Period:** 3 days
- **Action:** AI drafts follow-up email/WhatsApp to Promoter/Agent
- **Tone:** "Warm Professional" (75% Personal, 25% Logistics)

### Implementation

#### Option A: Google Calendar + Zapier/Make (No-Code)
1. Create a Google Calendar for "Kirk Whalum Shows"
2. Add events with venue name in title: "Show - [Venue Name] - [City]"
3. Use Zapier/Make to detect event end → wait 3 days → trigger email draft

#### Option B: OpenClaw Cron + Email (Recommended)
1. I'll check your Google Calendar daily for shows from 3 days ago
2. Generate personalized follow-up using your style guide
3. Notify you via message: "Draft ready for [Venue]. Review & Send?"
